# 🎓 University Portal

A responsive **University Portal** built using **HTML, CSS, and JavaScript** to simplify student registration and profile management. The application allows students to register, store their information locally, and view their profile through a clean, modern, and user-friendly interface.

---

## 📌 Features

- 📝 Student registration form with input validation
- 👤 Student profile display
- 💾 Local Storage for saving and retrieving student data
- 📱 Responsive and modern user interface
- 🕒 Dynamic date and time display
- ✅ User-friendly success and error notifications

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Browser Local Storage
